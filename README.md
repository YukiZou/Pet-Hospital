# pet-hospital
## 项目介绍
虚拟宠物医院学习系统是一个虚拟宠物医院教学软件，可以使得宠物工作者不去实体医院就能系统地学习各种宠物诊疗专业知识。针对相关专业毕业实习医生，提供了宠物医院结构、科室、病例学习等数据信息，同时为不同岗位角色配置了大量的真实病例。
## 功能介绍
* 用户管理
* 科室管理
* 试题管理
* 角色模块流程管理
* 病例管理
## 技术栈
* 前台框架：Vue.js
* 后台框架: SSM(Spring+SpringMVC+MyBatis）
* 数据库: MySQL
* Web服务器: Tomcat
* 编程语言: js + Java
